# Analysis
### MorseCode World
Utilisation du site morsecode.world pour le décodage.
https://morsecode.world/international/decoder/audio-decoder-adaptive.html
Comme le site affiche le spectogramme, on peut lire directement le flag:

**ADV{IsyourChristmastreeready?}**


### Audacity
Simplement, ouvrir le fichier audio.
il y a un petit menu sur la piste (cliquer sur '...'), et passer en vue spectogramme.
Et voilà le flag!

**ADV{IsyourChristmastreeready?}**
